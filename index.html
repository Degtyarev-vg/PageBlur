PageBlur

PageBlur - это кроссбраузерная библиотека для создания эффекта размытия страницы.
С помощью этой библиотеки можно размыть любой элемент на странице, и использовать полученое изображение для создания различных эффектов на сайте. PageBlur - it is cross-browser library for create at blur effect page.
With this library you can blur any element on the page, and use the image to create different effects on the site.
